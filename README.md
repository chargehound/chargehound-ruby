## Usage

Import chargehound and set your API key.

```ruby
require 'chargehound'
Chargehound.api_key = '{ YOUR_API_KEY }'
```

## Development

To build and install from the latest source:

```bash
$ git clone git@github.com:chargehound/chargehound-ruby.git
$ bundle install
```

Run tests:

```bash
$ bundle exec rake
```
